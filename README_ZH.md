# LayerAI

**本地优先的网页 AI 翻译工作流**

[English](./README.md) · [官网](https://layerai.yun/) · [文档](https://docs.layerai.yun) · [Chrome 扩展](https://chromewebstore.google.com/detail/enpnogogneglhpcdkfppcjnbbjafhikc)

---

LayerAI 是一款浏览器扩展，帮助你在网页上完成文本采集、AI 翻译、术语治理与多语言替换。

与每次打开页面都重新翻译不同，LayerAI 会把确认过的译文沉淀为可复用资产：同一页内容未变化时，已定稿译文会自动应用。

---

## 核心能力

| 能力 | 说明 |
|------|------|
| **采集** | 提取当前页、整站或选区文本 |
| **翻译** | 多模型 AI 翻译，支持术语表约束与队列重试 |
| **替换** | 在原网页自动应用已保存译文 |
| **治理** | 术语表与共享译文提交/审核，保持用词一致 |
| **同步** | 本地导入导出；Pro / Max 支持云同步与 Google Drive 备份 |

Free 计划即可走通完整流程；Pro / Max 提供更高额度、协作与同步能力。

---

## 快速开始

1. 安装 [Chrome 扩展](https://chromewebstore.google.com/detail/enpnogogneglhpcdkfppcjnbbjafhikc)
2. 打开任意外语网页，完成一次：采集 → 翻译 → 保存
3. 刷新页面，已保存译文会自动应用

更完整的引导见文档：[简介](https://docs.layerai.yun/introduction) · [第一次翻译](https://docs.layerai.yun/journey/first-translation)
